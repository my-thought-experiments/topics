# Envoy

## Installation

Assuming below items are aleady installed, up and running on your machine.

- [x] Node.js v18 or above.
- [x] Wireshark

## How to run examples?

Run Node.js app

```console
node index.mjs
```

Run Wireshark and Capture

Hit Curl
```
curl http://<IP>:3000 -v
```
